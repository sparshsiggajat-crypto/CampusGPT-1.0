## 📖 Overview

CampusGPT is an AI-powered Student Copilot designed to simplify campus life by providing students with instant access to academic information, institutional policies, attendance records, fee details, scholarships, hostel information, fines, and more through an intelligent conversational interface.

Instead of navigating multiple portals or searching through PDFs, students can simply ask questions in natural language and receive personalized, context-aware responses.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/CampusGPT-AI-Student-Copilot.git
```

Install frontend dependencies

```bash
npm install
```

Run frontend

```bash
npm run dev
```

Install backend dependencies

```bash
pip install -r requirements.txt
```

Run backend

```bash
python app.py
```
