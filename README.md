# 🎓 CampusGPT – AI Student Copilot

An **AI-powered Student ERP and Campus Assistant** that simplifies access to academic information through conversational AI. CampusGPT combines student records, institutional policies, and AI to provide personalized, accurate, and instant responses.

## 🚀 Live Demo

🌐 **https://campusgpt-ai-student-copilot-356200851822.asia-southeast1.run.app**

## ✨ Features

* 🤖 AI Chat Assistant (Google Gemini)
* 📄 AI Campus Policy Navigator (RAG)
* 👨‍🎓 Student Dashboard
* 📅 Attendance Management
* 💰 Fee Management
* 🏠 Hostel Management
* 🎓 Scholarship Tracking
* ⚠️ Fine Management
* 🔔 Smart Notifications
* 🎙️ Voice Assistant
* 👨‍💼 Admin Dashboard
* 📊 Reports & Analytics

## 🛠️ Tech Stack

**Frontend:** React, Vite, TypeScript, Tailwind CSS, ShadCN UI

**Backend:** Flask/FastAPI, REST APIs, JWT Authentication

**Database:** MySQL

**AI:** Google Gemini, LangChain, ChromaDB, Sentence Transformers, PyPDF

## ⚙️ Installation

```bash
git clone https://github.com/yourusername/CampusGPT-AI-Student-Copilot.git

cd CampusGPT-AI-Student-Copilot

# Frontend
npm install
npm run dev

# Backend
pip install -r requirements.txt
python app.py
```
